# Domínio de Gerenciamento de Solicitantes (gsolicitantes)

Este domínio é responsável pelo gerenciamento dos usuários solicitantes do sistema, incluindo:

- Cadastro, atualização e exclusão de solicitantes
- Listagem e busca de usuários
- Validação de dados dos usuários
- Visualização de chamados abertos pelo usuário

## Estrutura
- controller: Endpoints de solicitantes
- dto: Objetos de transferência de dados para solicitantes
- entity: Entidades relacionadas a solicitantes
- exception: Exceções específicas de solicitantes
- repository: Persistência de dados de solicitantes
- service: Lógica de negócio de solicitantes
